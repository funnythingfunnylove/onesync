# OneSync Privacy Policy

Effective date: July 2, 2026

OneSync is a browser extension for synchronizing bookmarks with a WebDAV server configured by the user.

## Data Handled by the Extension

OneSync may access and store the following data to provide bookmark sync:

- Browser bookmarks, including bookmark titles, URLs, folder structure, tags, and tag colors.
- WebDAV connection settings entered by the user, such as server URL, username, and password or app password.
- Local sync state, local snapshots, duplicate cleanup state, and activity logs.

## How Data Is Used

OneSync uses this data only to synchronize bookmarks, manage private bookmarks, remove duplicates, organize tags, and show sync status. The extension does not use this data for advertising, profiling, credit decisions, or any purpose unrelated to bookmark sync.

## Data Sharing

OneSync does not sell user data and does not transfer user data to third parties except as needed to sync with the WebDAV endpoint configured by the user. Bookmark data and WebDAV credentials are not sent to a OneSync-operated server.

## Remote Code

OneSync does not execute remotely hosted JavaScript or WebAssembly. Network access is used for communication with the user-configured WebDAV endpoint.

## User Control

Users control the WebDAV endpoint and can remove the extension or clear its stored extension data through the browser at any time.
